## Features
- Food menu display
- Interactive shopping cart
- Quantity controls
- Real-time price calculation
- Order summary
- Reset functionality

## Technologies Used
- HTML5
- CSS
- JavaScript
- jQuery

## How to Use
1. Open `index.html` in browser
2. Click "+" to add items to cart
3. Adjust quantity with +/- buttons
4. Click "Pesan" to checkout
5. View order summary
